# 🔧 Руководство по настройке устройств

## 📋 Проблема

Когда вы подключаете несколько контроллеров Maschine Mikro MK3, операционная система может находить их **в случайном порядке**:

```
Сегодня:
  Device 1 = Serial 0001A0B2 (левый контроллер)
  Device 2 = Serial 0001A177 (правый контроллер)

Завтра (после перезагрузки):
  Device 1 = Serial 0001A177 (правый контроллер) ← Поменялись местами!
  Device 2 = Serial 0001A0B2 (левый контроллер)
```

**Результат:** Путаница в играх и программах! 😕

---

## ✅ Решение: Постоянная привязка

**`device_setup.py`** - утилита для настройки постоянного порядка контроллеров.

### Как это работает:

1. Каждый контроллер имеет **уникальный серийный номер** (например: `0001A0B2`)
2. Вы **один раз** указываете какой контроллер = Device 1, 2, 3 или 4
3. Настройка сохраняется в `~/.maschine_device_config.json`
4. **Все игры и программы** автоматически используют эту настройку!

---

## 🚀 Быстрый старт

### Шаг 1: Запустите настройку

```bash
cd /Users/avk/p/maschine
python3 device_setup.py
```

### Шаг 2: Идентифицируйте контроллеры

Программа подсветит каждый контроллер **уникальным паттерном**:

```
====================================================================
🔍 Устройство: Устройство #1
   Serial: 0001A0B2
   Подсвечиваю 1 пэд...
====================================================================

[На контроллере загорается 1 пэд в верхнем левом углу]

Какой номер присвоить этому контроллеру? (1-4): _
```

**Паттерны идентификации:**
- **1 пэд** 🟦 = первое найденное устройство
- **2 пэда** 🟦🟦 = второе найденное устройство
- **3 пэда** 🟦🟦🟦 = третье найденное устройство
- **4 пэда** 🟦🟦🟦🟦 = четвёртое найденное устройство

### Шаг 3: Присвойте номера

Для каждого подсвеченного контроллера укажите **какой номер** вы хотите ему присвоить:

```
Какой номер присвоить этому контроллеру? (1-4): 1
✅ Serial 0001A0B2 → Device 1

====================================================================
🔍 Устройство: Устройство #2
   Serial: 0001A177
   Подсвечиваю 2 пэда...
====================================================================

[На другом контроллере загорается 2 пэда]

Какой номер присвоить этому контроллеру? (1-4): 2
✅ Serial 0001A177 → Device 2

... и так далее для всех контроллеров
```

### Шаг 4: Сохраните конфигурацию

```
====================================================================
📋 Итоговая конфигурация:
====================================================================
  Device 1 → 0001A0B2
  Device 2 → 0001A177
  Device 3 → 0001XXXX
  Device 4 → 0001YYYY

💾 Сохранить эту конфигурацию? (y/n): y

✅ Конфигурация сохранена: /Users/avk/.maschine_device_config.json
```

**Готово!** 🎉

## 🛠️ Для разработчиков

### Использование в своих программах:

```python
from maschine import find_devices, sort_devices_by_config

# Найти все устройства
devices = find_devices(max_count=4)

# Отсортировать согласно конфигурации
sorted_devices = sort_devices_by_config(devices)

# Теперь устройства всегда в правильном порядке!
for device, device_num in sorted_devices:
    print(f"Device {device_num}: {device.serial}")
    # Device 1: 0001A0B2
    # Device 2: 0001A177
    # ...
```

### Проверка наличия конфигурации:

```python
from maschine import has_config, get_config_path

if not has_config():
    print(f"⚠️  Конфигурация не найдена!")
    print(f"   Запустите: python3 device_setup.py")
else:
    print(f"✅ Используется конфигурация: {get_config_path()}")
```

## ❓ FAQ

**Q: Что если подключу меньше 4 контроллеров?**
A: Это нормально! Программы будут использовать только подключенные устройства в правильном порядке.

**Q: Что если подключу новый контроллер?**
A: Запустите `python3 device_setup.py` и добавьте его в конфигурацию.

**Q: Можно ли использовать разные конфигурации для разных программ?**
A: Нет, конфигурация общая для всех программ. Это сделано специально для единообразия.

**Q: Как удалить конфигурацию?**
A: Просто удалите файл:
```bash
rm ~/.maschine_device_config.json
```

**Q: Работает ли это на Windows/Linux?**
A: Да! Путь к конфигу будет:
- **macOS/Linux:** `~/.maschine_device_config.json`
- **Windows:** `C:\Users\YourName\.maschine_device_config.json`
